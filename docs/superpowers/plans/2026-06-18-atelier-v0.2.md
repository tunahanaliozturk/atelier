# Atelier v0.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Grow Atelier from an orchestration skeleton into a usable crew: add stack-layer implementation agents, make routing deterministic and testable, add resume/retry/crew commands plus an agent-scaffold skill, and harden validation.

**Architecture:** Markdown-driven plugin. Agents, commands, and skills are `.md` instruction files; `lib/` holds small, pure, `node:test`-covered ESM helpers. Claude performs orchestration by following the skills; the helpers make the mechanical parts (routing, mission validation) reproducible. The `SessionStart` hook surfaces the agent registry.

**Tech Stack:** Node.js (ESM `.mjs`), `node:test` + `node:assert` (no third-party deps), Markdown with YAML frontmatter.

## Global Constraints

- No third-party dependencies; helpers use only Node built-ins.
- Every agent `.md` frontmatter follows `docs/schemas/capability-schema.md`: `name`, `description`, `capabilities` (list), `layer`, `task_kinds` (list); optional `model`, `color`. Each frontmatter value stays on a single line.
- `layer` is one of: `orchestration`, `backend`, `frontend`, `infra`, `data`, `docs`, `review`.
- Mission status is one of: `pending`, `in-progress`, `blocked`, `done`.
- Task descriptions must never contain the pipe character `|` (table delimiter).
- `node --test` must stay green after every task.
- Commit after each task.

## File Structure

- `agents/{backend,frontend,infra,data}-engineer.md`, `agents/docs-writer.md`, `agents/security-reviewer.md` — new crew (Task 1).
- `lib/routing.mjs` + `lib/routing.test.mjs` — deterministic routing (Task 2).
- `lib/mission.mjs` + `lib/mission.test.mjs` — add `validateMission`, `setTaskStatus` (Task 3).
- `lib/registry.mjs` + `lib/registry.test.mjs` — add field validation (Task 4).
- `commands/crew.md` (Task 5), `commands/resume.md` (Task 6), `commands/retry.md` (Task 7).
- `skills/writing-agents/SKILL.md` + `skills/capability-routing/SKILL.md` edit (Task 8).
- `README.md`, `CHANGELOG.md`, `docs/schemas/capability-schema.md`, `docs/schemas/mission-schema.md` — docs alignment (Task 9).

Create `lib/routing.mjs`:

```js
// Deterministic routing helpers. Given a task descriptor and the agent registry
// (from readRegistry), filter to layer/kind-compatible agents and rank them by
// capability fit. The capability-routing skill leans on this for the mechanical
// stages; the lead still reads descriptions for the final nuance.

export function coarseFilter(task, agents) {
  return agents.filter(
    (a) =>
      (task.layer === undefined || a.layer === task.layer) &&
      a.task_kinds.includes(task.kind),
  );
}

export function scoreCandidate(task, agent) {
  let score = 0;
  if (agent.task_kinds.includes(task.kind)) score += 2;
  for (const cap of task.capabilities || []) {
    if (agent.capabilities.includes(cap)) score += 1;
  }
  return score;
}

export function route(task, agents) {
  const filtered = coarseFilter(task, agents);
  if (filtered.length === 0) {
    const where = task.layer ? ` in layer "${task.layer}"` : '';
    return {
      candidates: [],
      pick: null,
      reason: `no agent covers kind "${task.kind}"${where}`,
      fallback: true,
    };
  }
  const candidates = filtered
    .map((a) => ({ name: a.name, score: scoreCandidate(task, a) }))
    .sort((x, y) => y.score - x.score || x.name.localeCompare(y.name));
  const top = candidates[0];
  return {
    candidates,
    pick: top.name,
    reason: `picked ${top.name} (score ${top.score}) for kind "${task.kind}"`,
    fallback: false,
  };
}
```

```js
export const STATUSES = new Set(['pending', 'in-progress', 'blocked', 'done']);

export function validateMission(mission) {
  const tasks = mission.tasks || [];
  const ids = new Set();
  for (const t of tasks) {
    if (ids.has(t.id)) throw new Error(`duplicate task id: ${t.id}`);
    ids.add(t.id);
  }
  for (const t of tasks) {
    if (!STATUSES.has(t.status)) {
      throw new Error(`task ${t.id}: invalid status "${t.status}".`);
    }
    for (const d of t.deps || []) {
      if (!ids.has(d)) {
        throw new Error(`task ${t.id}: dep "${d}" is not a known task id.`);
      }
    }
  }
  detectCycle(tasks);
  return mission;
}

function detectCycle(tasks) {
  const byId = new Map(tasks.map((t) => [t.id, t]));
  const state = new Map(); // id -> 'visiting' | 'done'
  const visit = (id, stack) => {
    if (state.get(id) === 'done') return;
    if (state.get(id) === 'visiting') {
      throw new Error(`dependency cycle: ${[...stack, id].join(' -> ')}`);
    }
    state.set(id, 'visiting');
    for (const d of byId.get(id).deps || []) {
      if (byId.has(d)) visit(d, [...stack, id]);
    }
    state.set(id, 'done');
  };
  for (const t of tasks) visit(t.id, []);
}

export function setTaskStatus(mission, id, status) {
  if (!STATUSES.has(status)) throw new Error(`invalid status "${status}".`);
  const task = (mission.tasks || []).find((t) => t.id === id);
  if (!task) throw new Error(`no task with id "${id}".`);
  task.status = status;
  return mission;
}
```
